How to Run:


1. Create a new folder in your computer.

2. Download the zip file and extract it in the folder you created.

3. Open your VS CODE and open the file you created, it should contains folder and files you extracted.

4. Open your VS CODE terminal and type this "cd Area-51-Lab-Escape-main" and press ENTER, the current path of your terminal should be now look like this "C:\Temporary\Area-51-Lab-Escape-main>", type this again "cd Area-51-Lab-Escape-main" and ENTER, now the path of your terminal should be like this "C:\Temporary\Area-51-Lab-Escape-main\Area-51-Lab-Escape-main>"

5. if your path of your terminal is look like this "C:\Temporary\Area-51-Lab-Escape-main\Area-51-Lab-Escape-main>" then type this "pip install -r requirements.txt", you should see there a text like this "Requirement already satisfied: pygame in c:\python311\lib\site-packages (from -r requirements.txt (line 1)) (2.6.1)"

6. if step 5 is successful, you may now run the program by typing "python area51.py"
